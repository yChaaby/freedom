# freedom
## **Description**

Ce projet implémente un système de gestion d'utilisateurs et d'opinions en utilisant **Java RMI (Remote Method Invocation)**. 
Il permet aux utilisateurs de différents types (utilisateur régulier, influenceur, penseur critique, proposeur, chercheur de consensus)
d'interagir les uns avec les autres en proposant des sujets, en envoyant des opinions et en suivant d'autres utilisateurs.

## Exécution
Pour lancer le serveur:
java Main.java
pour lancer le client:
java client.java
pour lancer le polarimètre:
java Polarimeter.java
